# 大気光学3Dラボ / Atmospheric Optics 3D Lab

虹やハロが「なぜその角度に見えるのか」を、観察者・太陽・水滴・氷晶の3D関係と粒子内部の光路から確かめる、Nature Wx Labの教育用Webツールです。

公開URL: `https://nature-wx-lab.github.io/atmospheric-optics-lab/`

## 現在の公開範囲

開発中のβ版です。現在の公開版は次を実装しています。

- 虹画面の3つのカメラ操作: 虹から雨滴群へ進み、雨域到着後は自由探索し、人が選んだ雨滴へ連続的に寄る「雨域へ接近／選択滴へ接近」、ドラッグ／矢印で向きを変えてズーム操作を物理的な前後移動にする「観測者を移動」、切替時の観測条件と寄与計算を固定して3D構造を回り込む「光路を外から」
- 接近モードでは初期雨滴を自動選択しません。ホイール、ピンチ、range、ボタン、ドラッグは雨滴IDを選ばず、「画面の雨滴を選ぶ」を明示的に有効化した短いタップ、赤側656.3 nm・緑530 nm・青紫側404.7 nm、固定ID、前後候補だけが選択を作ります
- ホイール／2本指ピンチ／range／ボタンによる連続意味ズーム。未選択で雨域へ到着した後はホイール・ピンチ・WASDで前後左右、Q/Eで上下へ検査カメラを自由移動できます。「観測者を移動」では同じズーム操作が物理的な観測者の前進・後退になります
- 太陽条件とは独立した固定seed・永続ID付き60,000滴の代表雨滴場
- 一次虹・二次虹の観察者中心の円錐と、反太陽点からの角距離による全滴の寄与判定
- IAPWS R9-97による水の真空屈折率をCiddor標準乾燥空気の屈折率で割った水／空気相対屈折率と、代表7波長の幾何光学による角半径
- 球面との各交点・法線・入射方向からベクトル計算するSnell屈折、鏡面反射、部分透過、射出光路
- 水滴へ入る前は全可視波長が重なった白色光1本とし、空気―水境界の共通入射点から初めて代表7波長へ分散する表示
- 380–780 nmを4 nm間隔で積分し、5778 K黒体の相対太陽スペクトルとCIE 1931標準観測者を通して表示色へ変換する連続した遠景虹
- 波長別停留角を約0.53°の太陽円盤と代表的な雨滴径広がりの等価幅で平滑化する相対放射輝度近似。絶対輝度や完全なAiry／Lorenz–Mie計算ではありません
- 明示的な雨滴選択モードでの色付き・灰色を問わない全表示雨滴のクリック／タップ選択、赤・緑・青紫の寄与候補循環、重なり候補の循環、全60,000 IDの直接指定、前後の寄与滴を選べる代替操作
- 遠景の連続放射輝度、同じ角条件の代表雨滴群、選択した同じ実在ID、0.45–1.00 mmの等価球模型、1本の光路、7波長までをクロスフェードでつなぐ連続的な対数意味ズーム
- 寄与滴では視線角に対応する波長・屈折率を代表標本間で解き、目へ一致する停留光線を逐次表示
- 非寄与滴も拡大できる一方、比較用の虹光路を観察者の目へ接続しない明示的な非到達表示
- 内部反射面1面のFresnel反射率と、反射せず水滴外へ透過する光の枝
- 太陽高度・方位と雨滴描画密度の操作
- 主画面の固定60,000滴場内では、観測者を開始点から最大45 m動かし、同じ雨滴世界座標に対する全IDの寄与を移動後の観測点から再計算
- 主画面とは別に、固定した24,000滴の専用雨域を反太陽方向へ最大500 m移動する従来の「虹を追う」比較実験
- 移動前後の虹角度、寄与水滴数、共通する水滴ID、距離標本の比較
- 22°ハロ、46°ハロ、幻日、環天頂アーク、上部タンジェントアーク、環水平アーク
- 数式から求めた環状ハロ、投影近似、模式図を画面上で明確に区別
- 連続スケールrange、ホイール、2本指ピンチ、ボタン、キーボードを同じズーム状態へ統合
- スマートフォン縦・横の第一画面に主要操作と3Dを収め、必要時だけ描画する省負荷レンダリング
- PC・スマホ向けの密度の高い操作画面

氷晶内を通る光線の完全な3D追跡、Airy／Lorenz–Mieによる波動光学、絶対強度・偏光、太陽円盤の厳密な面積分、Monte Carlo光線追跡、雨滴の非球形状は今後の増築対象です。水滴光路は理想球の幾何光学として計算し、未計算の光路や現象を計算済みとは表示しません。

## 科学モデル

- 水滴は理想球、光は幾何光学として扱います。
- 60,000滴は大量の雨滴を理解するための代表標本で、実際の雨滴総数や降水量の推定値ではありません。各滴の位置とIDは太陽条件から独立し、太陽を動かしたときは寄与分類だけを更新します。
- 水の真空屈折率はIAPWS R9-97の式を20℃・密度998.2071 kg/m³で計算します。Snellの法則には、それをCiddorの標準乾燥空気分散式と実在気体の圧縮係数を用いて20℃・101325 Paへ密度補正した空気屈折率で割った、水／空気の相対屈折率を使います。乾燥空気はCO₂ 450 ppm固定で、湿度とCO₂濃度の変動は未実装です。
- 一次虹は内部反射1回、二次虹は2回です。いずれも全反射ではなく部分反射として説明します。
- 停留光線は、内部反射回数を `k` とした解析条件 `cos²(i) = (n² − 1) / (k(k + 2))` から求めます。
- 水滴の局所光路は、直線と単位球の交点ごとに外向き法線を求め、ベクトル形式のSnellの法則と `d′ = d − 2(d·N)N` の鏡面反射則で追跡します。表示する入射角・屈折角・反射角・射出角も、その交点の方向ベクトルと法線から計算します。
- 遠景の虹は個々の雨滴点ではなく、各視線方向へ届く波長別散乱を積分した連続相対放射輝度として描きます。380–780 nmを4 nm間隔で標本化し、水の屈折率はIAPWS由来の代表値間を補間、両端だけ近傍2標本から線形外挿します。入射光は5778 K黒体の相対スペクトル、色変換はCIE 1931の解析近似によるXYZからlinear sRGBです。
- 遠景の角度分布は、有限な太陽円盤と雨滴径分布が細かな構造を平滑化することを踏まえた等価ガウス幅で畳み込みます。自然虹の絶対輝度、偏光、過剰虹を再現する完全なAiry／Lorenz–Mieモデルではありません。
- 各代表雨滴について、観察者から水滴への方向と反太陽軸の角距離を求め、可視スペクトルの虹角範囲内にある滴を選択候補とします。この60,000滴は連続虹を構成する実際の全雨滴数ではなく、遠景から1滴へ降りるための決定論的な標本です。
- 接近モードの検査カメラは観察者の光学原点から始まり、反太陽方位へ約90°の水平視野を確保し、0〜47%では選択の有無にかかわらず同じ経路で雨域へ60 m進みます。到着後は固定観測点から260 m以内で自由探索できますが、物理的な観測者位置、寄与ID、波長判定、滴から目への光路は変えません。雨滴を人が選ぶまでは既定IDへ回頭せず、選択した後も47%を越えるまでそのIDはカメラ経路へ影響しません。
- 観測者移動モードでは、ドラッグ／矢印で水平方位を変え、ホイール、ピンチ、±操作で前後へ移動します。主雨滴場の世界座標とIDは固定したまま、観測者位置から全60,000 IDの視線角と寄与を再計算します。移動は80–300 m球殻標本の境界影響を抑えるため、開始点から45 m以内に制限します。
- 外側モードでは、切替時の物理観測者、太陽、虹次数、描画密度を計算スナップショットとして保持します。外側カメラを動かしても寄与判定は変えず、色点で全表示寄与ID、色線で最大48本の代表的な観測点―寄与滴結線を示します。
- 寄与滴では、雨滴の視線角と停留角が一致する屈折率を隣接する代表波長間で数値的に解きます。内部反射面1面のs偏光・p偏光別と無偏光平均のFresnel反射率を計算しますが、入射面・射出面の損失は含めず、虹全体の強度ではありません。
- 代表7波長は連続スペクトルを理解しやすく表示するサンプルであり、自然の虹が7本の光線に分かれているという意味ではありません。
- 分散比較では、選択波長の停留光線と同じimpact parameterを代表7波長へ共有します。そのため全波長の入射点は同一で、各屈折・反射経路はその波長とimpact parameterに対して正確ですが、選択波長以外の6本は各波長固有の停留光線ではありません。
- 水滴模型は直径0.45–1.00 mmの等価球です。大きな落下雨滴の扁平・非対称形状は未実装です。
- 別タブの従来版「虹を追う」は、主画面の60,000滴・自由移動とは別の固定24,000滴標本を使う再現可能な500 m比較実験です。角度がほぼ同じまま寄与する水滴群が変わることと、角度だけでは虹までの距離を一意に決められないことを示します。
- 22°・46°ハロは氷の六角柱プリズムの最小偏角を計算します。幻日は水平板状氷晶の投影近似、向きに依存する3種のアークは太陽高度条件を示す模式図です。
- ハロ画面の黄色い線は光路の案内模式で、氷晶内の屈折点・入射角・射出角を計算した光線追跡ではありません。
- 波動光学が必要な過剰虹、回折、干渉、偏光、絶対輝度は現βの対象外です。

## 連続ズームのモデル

虹は距離が一意に決まる物体ではないため、300 m先に固定した虹へ実寸カメラで接近する実装にはしていません。遠景では未分解の大量雨滴が作る連続相対放射輝度を表示し、接近時に80–300 mの代表雨域へ固定した60,000滴を徐々に解像します。未選択のまま雨域へ着くと、寄与計算とは独立した検査カメラでその3D空間を自由探索できます。その中から現在の観察者中心の波長別円錐帯へ入るIDを選択でき、ズーム対象は必ずこの固定配列に存在する滴です。初期状態に選択済みの雨滴はなく、選択前に既定IDへ引き寄せません。選択後も別の説明用水滴へ切り替えません。

物理的な画角ではなく、学習段階を表す意味スケール指標を約300 m相当から350 µm相当まで対数補間します。0〜47%は全利用者に共通する虹から雨滴群への経路、47%以降は選択直前の検査カメラ姿勢から同じIDへの連続接近、同じ位置に重ねた局所水滴模型、内部光路を見やすい側面視点までをつなぎます。ID・世界座標・固定観察者への方向は往復しても変えません。kmとmmを同じThree.js座標の実寸として扱わない「意味ズーム」であり、画面寸法から校正した実測縮尺ではありません。

表示段階:

1. 未分解の大量雨滴から積分した連続相対放射輝度
2. 同じ虹角を共有する固定60,000滴の代表雨滴場と寄与候補ID
3. 固定観測者の寄与計算を変えずに移動できる、雨滴群内の自由探索カメラ
4. 赤・緑・青紫の寄与候補、明示選択モードのクリック／タップ、重なり候補、ID指定、前後ボタンのいずれかで人が選んだ同じ実在ID
5. 選択滴の0.45–1.00 mm等価球模型
6. 寄与滴では目へ届く停留光線、非寄与滴では目を外れる比較光線
7. 代表7波長と部分透過枝

## ローカル起動

Node.js 24とpnpm 11を使用します。

```bash
pnpm install --frozen-lockfile
pnpm dev
```

検証:

```bash
pnpm test
pnpm build
node scripts/verify-build.mjs
python3 scripts/privacy_gate.py
```

## 公開安全設計

このリポジトリだけを独立した公開正本とし、親の非公開作業場、参考動画、スクリーンショット、ローカルデータ、個人情報をコピーしません。

公開前にローカルで、さらにGitHub Actionsを防御の二段目として次を検査します。

- 現行ファイルと到達可能な全Git履歴
- author / committerのnoreply allowlist
- commit messageと過去のファイル名・blob
- ローカル絶対パス、メール、秘密値・credential候補
- 動画、スクリーンショット、画像、PDF、DB、表計算、source map、symlink
- Pages配信allowlist
- `deployment.json`のsource commitと全配信ファイルSHA-256

push前に全履歴privacy gateを必ず合格させ、公開リポジトリへ不適切な履歴を送らないことを第一の防止策にします。GitHub Actions側でも同じ検査を再実行し、失敗時はPages artifactを作成せず配信更新を停止します。
