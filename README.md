# 大気光学3Dラボ

## 状態

このプロジェクトは、当初求めた「遠景の虹から多数の雨滴、個々の雨滴内部の光路までを、同じ3D空間と厳密な物理計算で連続的に探索できる教材」を、現在の実装では十分な品質で実現できないと判断し、2026年7月23日に開発を停止しました。

現行ブランチにはシミュレーター本体、物理計算コード、テスト、Node.js依存関係を残していません。公開URLも停止案内だけを配信します。

- 公開URL: `https://nature-wx-lab.github.io/atmospheric-optics-lab/`
- 最終の稼働版コミット: `44f67d322f580bc76184fb79c0f573d67c1e3457`
- 過去の実装はGit履歴にのみ残します
- 新しい要求仕様と受入基準を人が決めるまでは、自動的に再開しません

## 最低限残すもの

- この停止記録
- 停止を知らせる静的な公開ページと検索除外設定
- 公開前の個人情報検査、配信物のチェックサム作成、公開一致検証
- Git履歴による復旧手段
